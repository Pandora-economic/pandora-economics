# Dossier des images de couverture

Déposez ici les images de couverture des analyses (optionnel), par exemple :
- analyse-07.jpg
- analyse-06.jpg

Puis renseignez le champ `image` dans publications.js, ex. :
  image:"images/analyse-07.jpg"

Si aucune image n'est fournie, la page d'article affiche automatiquement
une couverture avec le numéro de l'analyse.
